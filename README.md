# Clínica Saúde Plena

Landing page institucional da **Clínica Saúde Plena**, desenvolvida com HTML e CSS puros, sem dependências externas.

**Demo ao vivo:** https://vitoriafspaiva.github.io/saude-plena/

---

## Seções

- **Hero** — chamada principal com botão de agendamento
- **Serviços** — cards com Atendimento Médico, Exames Laboratoriais, Nutrição Clínica e Procedimentos Especializados
- **Equipe** — perfis dos profissionais de saúde
- **Contato** — formulário para agendamento de consultas

## Tecnologias

- HTML5 semântico
- CSS3 (Grid, Flexbox, variáveis de cor, media queries)
- Layout responsivo para mobile e desktop

## Como rodar localmente

```bash
# Basta abrir o arquivo no navegador
open index.html
```

Ou use qualquer servidor estático:

```bash
npx serve .
# ou
python3 -m http.server 8080
```

## Estrutura

```
saude-plena/
└── index.html   # página única (HTML + CSS inline)
```
